# MIPS32-logisim
## 本项目文件结构
### 流水线部分-单周期.circ
MIPS 32位单周期流水线，支持28条指令，已扩展cp0协处理器，但异常和中断功能未完善
### 流水线部分-多周期.circ
MIPS 32为多周期5级流水线，支持25条指令，数据和控制冒险均已处理
### 存储部分-cache.circ
直接映射cache，但是未添加进上面流水线的设计中
### 存储部分-主存RAM.circ
上面流水线使用的存储器设计，支持三种访问方式，分别为半字、字节和字访问
